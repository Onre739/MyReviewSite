About this project:
Tento projekt vznikl za účelem seznámení se s Spring frameworkem a také k zužitkování znalostí získaných studiem na VŠB. Jde o informační systém pro recenzování her a jejich dlc.
V projektu byl použito:
  • Bootstrap - vylepšení vizuálu
  • Framework Spring, Spring Boot
  • SQLite databáze
  • Maven
  • Hibernate - ORM
  • Hosting zdarma přes render.com
  • Nasazení pomocí Docker
  • Git/GitHub - verzování
